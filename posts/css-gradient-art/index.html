CSS 中的渐变（gradient）有许多种形态，针对其语法与能实现的效果做一次系统性的整理。

<gradient> 包括：

  1. Linear gradient（线性渐变）
  2. Radial gradient（径向渐变）
  3. Repeating gradient（重复渐变）
  4. Conic gradient（锥形渐变）

1. Linear gradient

线性渐变，最常用的渐变形态。
